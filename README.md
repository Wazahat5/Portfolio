# Mohammad Wazahat Khan 3D Portfolio Website

This is a complete 3D animated portfolio website using:

- HTML
- CSS
- JavaScript
- Your profile photo

## How to run

1. Extract the ZIP file.
2. Open `index.html` in Chrome or Edge.
3. Edit your email, Fiverr link, skills, and project names inside `index.html`.

## Files

- `index.html` - website structure
- `style.css` - full 3D design and responsive styling
- `script.js` - 3D tilt, stars animation, menu, reveal animations
- `assets/profile.jpg` - your profile photo
